# Skylight Landing Page

High-performance landing page with scroll-triggered animations, responsive design, and conversion-optimized layout.

## Features

- Smooth scroll animations with IntersectionObserver
- Mobile-first responsive design
- Conversion-optimized CTA sections
- Lazy-loaded images for performance
- Contact form with validation

## Tech Stack

JavaScript, HTML, CSS

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/skylight-landing-page.git
cd skylight-landing-page
Open index.html in a browser.
```

### Usage

```bash
Open index.html in any modern browser.
```
